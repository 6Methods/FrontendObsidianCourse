# Модуль 02 — HTML

**roadmap.sh:** [раздел HTML во frontend-roadmap](https://roadmap.sh/frontend#html)

HTML — это скелет любого сайта. Пока у нас нет ни стилей, ни JS — но уже можно сделать рабочую страницу, которую откроет любой браузер в мире. Этот модуль — фундамент, на который дальше лягут CSS и JavaScript.

## Что изучаем
- Структура HTML-документа: DOCTYPE, `<head>`, `<body>`, meta-теги, подключение CSS/JS.
- Основные теги контента: заголовки, текст, списки, ссылки, изображения.
- Семантика HTML5: `header`, `nav`, `main`, `section`, `article`, `footer` — и зачем они нужны.
- Формы: `input`, `label`, `button`, базовая HTML-валидация.
- Основы доступности (a11y): `alt`, `aria-label`, связь `label ↔ input`, клавиатурная навигация.
