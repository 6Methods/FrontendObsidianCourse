# Модуль 05 — JS база

**roadmap.sh:** [раздел JavaScript во frontend-roadmap](https://roadmap.sh/frontend#javascript)

JavaScript — язык, который оживляет страницу: реакция на клики, валидация форм, обновление DOM без перезагрузки. Этот модуль закладывает фундамент: синтаксис, типы, функции, массивы, объекты, DOM API. Без `this`, замыканий и асинхронности — это всё в модуле 06. Здесь задача попроще: научиться писать логику, которая решает задачу на странице.

## Что изучаем
- Переменные (`let`/`const`), примитивные типы, операторы, преобразования типов.
- Условные конструкции (`if`/`else`/`switch`), тернарный оператор, циклы.
- Функции: declaration vs expression, стрелочные, параметры, return, scope.
- Массивы и ключевые методы: `map`, `filter`, `reduce`, `find`, `forEach`.
- Объекты: литералы, доступ, деструктуризация, spread, JSON.
- DOM API: `querySelector`, `classList`, обработчики событий, создание элементов.
