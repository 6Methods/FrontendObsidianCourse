# Прогресс и ретро

Сюда пишутся заметки после каждой крупной вехи (P1, P2, P3) и раз в 1–2 модуля.

## Формат записи

```
### YYYY-MM-DD — [что закрыли]
**Что зашло:** …
**Что было сложно:** …
**Что хочется глубже:** …
**План на следующий блок:** …
```
