# Модуль 12 — Build tools и тестирование

**roadmap.sh:** [Frontend roadmap — Build tools + Testing](https://roadmap.sh/frontend)

Ты уже умеешь писать React+TS-приложения, но «работает у меня локально» — это не продакшн. Нужны инструменты, которые превратят проект в надёжный релиз: быстрый сборщик, линтер с разумными правилами, автотесты (юнит, компонентные, E2E) и CI, который не пустит в main красный код. Это финальный прод-слой перед Next.js и деплоем.

## Что изучаем
- Vite в деталях: dev-сервер, HMR, env, алиасы, прокси, build.
- ESLint + Prettier: разделение ролей, конфиг, интеграция с VS Code.
- Линт для TS+React: правила `typescript-eslint`, `react-hooks/exhaustive-deps`, jsx-a11y.
- Vitest: юнит-тесты для утилит и хуков.
- Testing Library: компонентные тесты по поведению, а не имплементации.
- Playwright: E2E поверх реального браузера, скриншоты и traces.
- CI на GitHub Actions: зелёная планка на PR, матрица, артефакты.
