# Модуль 14 — Деплой и финал

**roadmap.sh:** [Frontend roadmap](https://roadmap.sh/frontend) — финальные узлы «Deployment», «Performance», «Security», «Monitoring».

Ты написал fullstack-приложение. Но «работает у меня локально» — не работает в проде: нет HTTPS, нет CDN, картинки медленные, ошибки никто не видит, бот заспамил форму, и Lighthouse 45 из 100. Этот модуль превращает твой код в продуктовый продукт: деплой на Vercel, оптимизация Core Web Vitals, кеширование на edge, monitoring ошибок, безопасность в проде. И финальная часть — как упаковать курс в резюме и портфолио, чтобы тебя взяли.

## Что изучаем
- **Vercel и деплой**: GitHub-интеграция, preview deployments, env vars, rollbacks, `vercel.ts`.
- **next/image и next/font**: оптимизация LCP/CLS из коробки.
- **Метрики и Lighthouse**: LCP, INP, CLS, Speed Insights, как читать отчёт.
- **Кеш и edge**: Data/Router Cache, ISR, Fluid Compute в 2026.
- **Monitoring**: Vercel Analytics, Sentry, логи, уведомления.
- **Безопасность в проде**: CSP, CORS, rate limit, BotID, RLS, OAuth-редиректы.
- **Accessibility 2.0 и i18n**: WCAG 2.2, ARIA, focus management, next-intl.
- **AI-инструменты и code review**: Claude Code / Cursor / Copilot, PR-гигиена, review-диалог.
- **Финал**: как оформить портфолио, резюме, LinkedIn, что учить дальше.
