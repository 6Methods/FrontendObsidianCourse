# P2 — React SPA

> **Статус: заглушка.** Полное описание появится после готовности модулей 09–10. Эта страница нужна, чтобы wiki-link'и в курсе не были битыми.

## Когда

Второй крупный проект курса. Берётся после модулей **09 React-база** и **10 React-экосистема**.

## Что это будет

Single Page Application: одностраничное приложение на React с роутингом, асинхронными данными, формами и валидацией.

**Стек проекта:**
- React 19
- React Router (SPA-навигация)
- TanStack Query (асинхронные данные, кеш)
- React Hook Form + Zod (формы и валидация)
- Tailwind CSS (стили)
- Публичный API (на выбор: PokéAPI, Rick & Morty, RAWG, JSONPlaceholder)

**Деплой:** Vercel — здесь он уже логичен, потому что есть сборка через Vite и SPA-роутинг.
