# Модуль 00 — Установка

**roadmap.sh:** [Frontend Roadmap — начало пути](https://roadmap.sh/frontend)

Это нулевой, подготовительный модуль. Не про код — про то, чтобы твой компьютер был готов к обучению. Три урока, каждый с конкретными шагами: поставил, проверил, пошёл дальше.

Если ты дизайнер и никогда не открывал терминал — это именно то место, где мы берём тебя за лапку и аккуратно всё объясняем.

## Что изучаем

- Список инструментов: зачем каждый из них существует.
- VS Code — установка, базовая настройка, нужные расширения.
- Node.js, npm и терминал — установка + первые команды, чтобы не пугаться чёрного экрана.

- [ ] Расширения Prettier и ESLint установлены.

- [ ] Создан аккаунт на GitHub.
