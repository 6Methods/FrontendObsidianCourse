# Flexbox

## Зачем это нужно

Flexbox — главный инструмент вёрстки последних десяти лет. Он решает базовую задачу любого интерфейса: «расположи элементы в ряд или колонку и нормально распредели между ними пространство». Если посмотреть на современные сайты, то примерно 80% интерфейсов — это flex-контейнеры, вложенные в другие flex-контейнеры, и без этого инструмента сегодня никуда.

## Главное

### Модель — контейнер и дети

Flexbox работает в паре «родитель плюс его дети». Ты пишешь `display: flex` **на родителе**, и он становится flex-container. Все его прямые дети автоматически превращаются в flex-items и начинают подчиняться правилам раскладки, которые ты задаёшь на контейнере. На внуков это не распространяется — если нужен flex глубже, делаешь вложенный контейнер.

### Оси

У flex-контейнера две оси, и почти все свойства привязаны к ним. **main axis** — основная ось, вдоль которой раскладываются items. По умолчанию она горизонтальная и идёт слева направо. **cross axis** — ось, перпендикулярная main axis, то есть по умолчанию вертикальная. Направление main axis меняется через `flex-direction`: `row` (по умолчанию), `row-reverse`, `column`, `column-reverse`. Как только ты поставил `flex-direction: column`, «главное» становится вертикалью, и все свойства начинают работать относительно неё.

### Свойства контейнера

- `justify-content` — как разложить items по main axis. Варианты: `flex-start` (прижать к началу), `center`, `flex-end` (к концу), `space-between` (крайние по краям, между остальными равные промежутки), `space-around`, `space-evenly`.
- `align-items` — как разложить items по cross axis. Варианты: `stretch` (по умолчанию, растягивает items на всю высоту контейнера), `center`, `flex-start`, `flex-end`, `baseline` (выравнивание по базовой линии текста).
- `flex-wrap: nowrap | wrap` — разрешать ли перенос на новую строку, когда items не помещаются в ряд. По умолчанию `nowrap`, и items будут сжиматься.
- `gap` — расстояние между items. Современный и единственно правильный способ делать отступы внутри flex-контейнера, заменяет старые костыли с `margin` у детей.
- `align-content` — как распределить **строки** по cross axis, когда их несколько (то есть когда сработал `flex-wrap`). На однострочный layout не влияет.

### Свойства item-а

- `flex: 0 1 auto` — значение по умолчанию. Это сокращение от трёх свойств: `flex-grow flex-shrink flex-basis`.
- `flex: 1` — самая частая запись. Значит «забери всё свободное пространство». Если поставить нескольким items — они поделят пространство поровну.
- `flex-basis` — стартовая ширина item-а до того, как его начнут растягивать или сжимать.
- `align-self` — перебивает `align-items` контейнера для одного конкретного item-а. Удобно, когда один элемент должен выравниваться иначе, чем остальные.
- `order` — меняет визуальный порядок items без перестановки HTML. Полезно для адаптива.

### Типичные шаблоны

- **Ряд с элементом справа.** `justify-content: space-between` плюс `gap` — классическая шапка сайта: логотип слева, навигация справа.
- **Центр по обеим осям.** `display: flex; justify-content: center; align-items: center;` — та самая «центровка div-а», про которую мемы.
- **Колонка, все на всю ширину.** `display: flex; flex-direction: column; gap: 16px;` — вертикальный список блоков с одинаковыми отступами.

## Пример кода

html:
```html
<header class="site-header">
  <div class="logo">MySite</div>
  <nav class="nav">
    <a href="/">Главная</a>
    <a href="/about">О нас</a>
    <a href="/contact">Контакты</a>
  </nav>
</header>
```

css:
```css
.site-header {
  display: flex;
  justify-content: space-between;
  align-items: center;
  gap: 16px;
  padding: 16px 24px;
}

.nav {
  display: flex;
  gap: 24px;
}

/* центрирование карточки */
.hero {
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 60vh;
}
```

## Частые ошибки

- Ставить `display: flex` на сами items вместо родителя → ничего не работает, потому что flex раскладывает детей, а не сам себя.
- Использовать `margin: auto` между items, когда есть `gap` → современно всегда `gap`, `margin` оставь для внешних отступов.
- Забывать `flex-wrap: wrap` → на маленьком экране items начинают выдавливать друг друга и ломать дизайн.
- Путать `justify-content` и `align-items` → запомни раз и навсегда: justify = main axis, align = cross axis.
