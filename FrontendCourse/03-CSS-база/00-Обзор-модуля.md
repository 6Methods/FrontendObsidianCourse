# Модуль 03 — CSS база

**roadmap.sh:** [раздел CSS во frontend-roadmap](https://roadmap.sh/frontend#css)

CSS превращает сырой HTML в то, на что хочется смотреть. Этот модуль — фундамент: как подключить стили, как выбрать нужные элементы селекторами, как работает каскад, box model и самое важное для вёрстки — Flexbox и Grid. К концу модуля ты сможешь сам сверстать адаптивный лендинг.

## Что изучаем
- Подключение CSS и базовый синтаксис правила.
- Селекторы: тег, класс, id, атрибут, комбинаторы, псевдо-классы/элементы.
- Каскад, специфичность, наследование — как браузер решает, какое правило применить.
- Box model (content / padding / border / margin) и позиционирование (`static`, `relative`, `absolute`, `fixed`, `sticky`).
- Flexbox — главный инструмент для одномерной раскладки.
- Grid + медиа-запросы — двумерная раскладка и адаптив под мобильные.
