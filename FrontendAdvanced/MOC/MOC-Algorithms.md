# MOC — Algorithms

Карта знаний Track 5: алгоритмы и структуры данных не ради LeetCode, а ради применимости на фронтенде. Пять модулей — сложность, структуры, деревья, графы, DP. Каждый урок — задача, которую ты реально встретишь в проде: autocomplete, LRU-кеш, топологическая сортировка зависимостей, VDOM diff, fuzzy search.
